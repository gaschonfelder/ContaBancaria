# ContaBancaria
## Descrição
Código desenvolvido no Eclipse IDE com o objetivo de executar seu teste e documentação com o auxilio da ferramenta Cucumber. Esse codigo foi realizado na linguagem Java, e feito junto com os recursos: Cucumber e JavaDoc.
